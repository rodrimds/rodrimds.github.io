Portafolio

Proyecto 1: Sitio web de comercio electrónico

Desarrollé un sitio web de comercio electrónico completo utilizando React y Node.js. El sitio incluye características como carrito de compras, pasarela de pago y gestión de inventario.

Proyecto 2: Aplicación de gestión de tareas

Creé una aplicación de gestión de tareas utilizando Angular y Firebase. La aplicación permite a los usuarios crear, editar y eliminar tareas, así como establecer recordatorios y fechas límite.

Proyecto 3: Blog personal

Diseñé y desarrollé un blog personal utilizando HTML, CSS y JavaScript. El blog incluye un sistema de comentarios, categorías y un diseño responsivo para una mejor experiencia en dispositivos móviles.
